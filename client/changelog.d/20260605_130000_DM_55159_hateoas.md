### New features

- Added optional `edition_url` to `EditionUpdateRef` and `queue_job_url` to `PublishJobRef` in `docverse.client.models`. These surface the HATEOAS links now embedded in a `build_processing` job's `progress` payload, so consumers can follow `edition_url` / `queue_job_url` instead of reconstructing `…/editions/{slug}` and `/queue/jobs/{id}` paths.
